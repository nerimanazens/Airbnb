# Airbnb Marketplace Learning Journey

## What I Learned

### React Hooks
- useState for local component state
- useEffect for side effects
- useMemo and useCallback for performance optimization
- useContext for global state access

### React Context
- ThemeContext for dark/light mode
- BookingContext for multi-step booking flow
- SearchContext for real-time property filtering
- WishlistContext for favorite properties

### Custom Hooks
- useFetch - API data fetching with loading/error states
- useLocalStorage - persisting data in browser storage
- usePropertySearch - property search with filtering
- useBookingForm - multi-step booking management

### Performance Optimization
- useMemo for expensive filter calculations
- useCallback for function memoization
- React.memo for component memoization

## Challenges

- Understanding prop drilling and why Context solves it
- Managing state across multiple components
- TypeScript type definitions for complex data structures
- Leaflet map integration with Next.js

## Key Takeaways

Context API eliminates prop drilling and makes global state management clean and maintainable. Custom hooks make logic reusable across components without code duplication.
